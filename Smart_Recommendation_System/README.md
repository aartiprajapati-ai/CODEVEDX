# 🎬 Smart Movie Recommendation System

## 📌 Overview

The Smart Movie Recommendation System is a Machine Learning project that recommends similar movies based on the user's selected movie. It uses Content-Based Filtering with CountVectorizer and Cosine Similarity.

---

## 🚀 Features

- Select a movie from the dropdown
- Get Top 5 similar movie recommendations
- Content-Based Recommendation
- Streamlit Web Application
- Machine Learning Model

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit

---

## 📂 Dataset

TMDB 5000 Movies Dataset

Files:
- tmdb_5000_movies.csv
- tmdb_5000_credits.csv

---

## 📊 Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Text Vectorization
5. Cosine Similarity
6. Recommendation Generation

---

## ▶️ Run Project

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py
