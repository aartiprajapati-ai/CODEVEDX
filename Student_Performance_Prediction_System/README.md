# Student Performance Prediction System

## Project Overview
This project uses Machine Learning to predict student final marks based on study hours, attendance and previous marks.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Streamlit

## Machine Learning Algorithm
Linear Regression

## Features
- Student data analysis
- Data visualization
- Performance prediction
- User-friendly Streamlit interface

## Input Features
- Study Hours
- Attendance
- Previous Marks

## Output
Predicted Final Marks

## How to Run
Install requirements:

pip install -r requirements.txt

Run application:

streamlit run app.py
