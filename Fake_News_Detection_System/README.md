# 📰 AI Based Fake News Detection System

## 📌 Project Overview

This project is an AI-powered Fake News Detection System developed using Machine Learning.

The system predicts whether a news article is Fake or Real using Natural Language Processing (NLP) and Logistic Regression.

---

## 🚀 Features

- Detect Fake and Real News
- NLP Text Cleaning
- TF-IDF Vectorization
- Logistic Regression Model
- Interactive Streamlit Web App
- Fast Prediction
- User Friendly Interface

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit
- Joblib
- Matplotlib
- Seaborn

---

## 📂 Dataset

Dataset contains two CSV files:

- Fake.csv
- True.csv

## 🤖 Machine Learning Algorithm

- Logistic Regression
