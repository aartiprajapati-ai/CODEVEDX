# 🚀 CodeVedX AI/ML Internship Projects

This repository contains all AI/ML projects completed during the CodeVedX Internship.

## 📂 Projects

### 1. Student Performance Prediction System
- Predicts student performance using Machine Learning.
- Technologies: Python, Pandas, Scikit-learn, Streamlit

### 2. Fake News Detection System
- Detects whether a news article is Fake or Real.
- Technologies: NLP, TF-IDF, Logistic Regression, Streamlit

### 3. Smart Movie Recommendation System
- Recommends similar movies using Content-Based Filtering.
- Technologies: Python, Pandas, Scikit-learn, Streamlit
